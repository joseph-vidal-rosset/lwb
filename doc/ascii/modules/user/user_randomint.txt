
randomint
=========

random integer

SYNTAX
- randomint(M, N) 

PARAMETERS
- M, N: natural numbers,  M <= N 

RESULT:  natural number

SYNOPSIS

- randomint  returns a natural number. For the result  R  we have  M <= R <= 
  N .

EXAMPLE

> randomint(1, 20);
	19
> randomint(1, 20);
	19
> randomint(1, 20);
	14
